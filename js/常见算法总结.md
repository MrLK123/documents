### 常见算法总结

**1. 冒泡排序：**

*思想：*

每次对比相邻两个数的大小，小的排在前面，若后比前大则交换两者位置大的置前。

优点简单易理解，缺点比较次数多，效率低。

```js
// 新增newArr是为了不影响原始数组
function bubbleSort(arr, newArr = []) {
	console.time('冒泡排序耗时');
	for(let i = 1; i < arr.length; i++){
		for(let j=0; j<=arr.length-i; j++){
      		// ">" 从小到大排序
      		// "<" 从大到小排序
      		if(arr[j] > arr[j+1]){
		        [newArr[j], newArr[j+1]] = [newArr[j+1], newArr[j]]
		    }
      	}
	}
	console.timeEnd('冒泡排序耗时');
	return newArr = arr;
}
```
